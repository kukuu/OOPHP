PHP/MySQL
PHP/MySQL  is a  popular  packaged  web programming   language  used for   developing wide range of web, database and eCommerce applications. 
 
PHP is an open source, interpretive, HTML centric, server side scripting language.It is especially suited for Web development and can be embedded into HTML pages.
 
MySQL  is an  open source programming language that can be used to create database applications for the web.
 
Extensive  hands-on exercise will allow students to write dynamic Web pages using database content. Basic knowledge and experience of html is required. Programming experience in the following languages is helpful but not necessary.JavaScript, ASP,VB and Perl.
 
You will learn to
  • Create database applications using MySQL  functions and commands
  • Build PHP Language  blocks
  • Exploit the functinality of PHP and Structure PHP code
  • Write Web pages with PHP  
    • Interacting with the server : Creating and Managaing database connections
    • Sending queries to the database server : Interacting with the server
    • Retrieving and displaying query results : Manipulating User input
    • Dealing with quotes and backlsashes in data
    • Error Handling
